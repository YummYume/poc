# POC

POC du 15/03/2021

Le projet consiste à réaliser un site sur lequel un utilisateur peut se connecter et ajouter les séries qu'il regarde.
Ce projet a été réalisé en ~6h à partir d'une maquette et d'un cahier des charges.

L'utilisateur peut :
- S'inscrire,
- Se connecter,
- Ajouter une série,
- Supprimer une série.

Lors de l'ajout d'une question, il peut : mettre un nom, une description, une vignette, la date de sortie, l'épisode et la saison actuelle, la catégorie et s'il a terminé de regarder la série.

Je n'ai pas eu le temps de faire : la modification d'une série et les fixtures.
